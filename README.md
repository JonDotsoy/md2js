# md2js


> ## Install

Install with [npm](https://www.npmjs.com/)

```sh
$ npm i md2js --save-dev
```

## Usage

```js
var md2js = require('md2js');
```

## Running tests

Install dev dependencies:

```sh
$ npm i -d && npm test
```
